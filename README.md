# 📚 Lexicon — персональный словарь для изучения иностранных языков

Современное веб-приложение для изучения языков через собственные коллекции слов.

Пользователь может сохранять незнакомые слова, распределять их по коллекциям и запускать квизы для тренировки памяти. Приложение анализирует прогресс обучения и помогает понять, какие слова уже запомнились, а какие стоит повторить.

---

# ✨ Возможности

## 📖 Личный словарь

* Добавление слов и переводов
* Примеры использования
* Заметки и комментарии
* Коллекции слов
* Поиск и фильтрация

## 🧠 Система квизов

* Перевод → слово
* Слово → перевод
* Смешанный режим
* Настройка количества слов
* Подсказки во время прохождения
* Мгновенная проверка ответов

## 📊 Статистика обучения

* Процент правильных ответов
* Аналитика запомненных слов
* Отслеживание прогресса

## 🎯 Система знания слов

Каждое слово получает статус на основе результатов квизов:

* ✅ Запомнил
* 🟢 Знаю хорошо
* 🟡 Нужно повторить
* 🔴 Не знаю
